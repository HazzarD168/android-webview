Product Description

Test your websites with Android WebView, using a configurable list of sites and matches to allow or skip. Other links will not be handled by the app and will simply ask you to open the standard browser or not.

Source code is available at https://github.com/mikespub/android-webview for inspiration and further customization.

Product Features

  • Predefined list of websites
  • Acceptable matches
  • Links to skip
  • Configurable settings
  • Editable settings.json file
  • Save state between sessions
  • Advanced configuration options
  • Easy access to test local websites
  • Download site bundle from trusted site
  • Support deep links to local sites
  • Test intent handling in demo site
